\# AI Skill Navigator for Moodle



AI Skill Navigator is a Moodle local plugin designed to support personalised learning paths for digital skills such as Artificial Intelligence, IoT, Digital Twin and Virtual Worlds.



\## Thesis idea



The goal of this project is to develop a Moodle plugin that integrates Artificial Intelligence to support:



\- student skill profiling;

\- personalised learning recommendations;

\- AI-based tutoring;

\- quiz and micro-exercise generation;

\- Virtual Worlds scenario generation;

\- teacher dashboards for skill gaps and learning analytics.

This project is designed as a Moodle local plugin:
