# Tesi Moodle AI

Repository per la tesi triennale sull'integrazione di strumenti di Intelligenza Artificiale in Moodle.

## Obiettivo

Progettare e sviluppare un prototipo sperimentale integrato in Moodle per supportare attività formative su AI, IoT e Digital Twin.

## Ambiente locale

Per avviare Moodle:

docker compose up -d

Per fermare Moodle:

docker compose down

Accesso Moodle:

http://localhost:8080

Credenziali locali:

username: admin
password: Admin123!

## Contesto

Il progetto di tesi studia come integrare strumenti di Intelligenza Artificiale in Moodle per supportare percorsi formativi legati a digital skills, AI, IoT e Digital Twin.
