# Provider Configuration

The AI provider is selected through Moodle plugin settings.

Supported provider names:

- `ollama`;
- `openai`;
- `openai_compatible`;
- `openrouter`;
- `prototype`;
- `mock`;
- `demo`.

Default values:

- provider: `ollama`;
- endpoint: `http://host.docker.internal:11434`;
- model: `qwen2.5:3b`.
